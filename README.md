# darkmod

This is a simple template for the usage of Dobby with a Mod Menu written in Java and C++

### Not For Noobs

### REQUIRED

1. A Brain
2. Java Skill ( Intermediate )
3. C++ Skill ( Intermediate )
4. Smali Skill ( Basic )
5. Apktool
6. Il2CppDumper
